dependencies:
    typescript -> Adds static typing to JavaScript and compiles .ts files into .js
    @types/node -> Provides TypeScript type definitions for Node.js modules
    ts-node -> Runs TypeScript files directly without manual compilation
    nodemon -> Automatically restarts the application when source files change
    prettier -> Automatically formats code according to consistent style rules
    eslint -> Analyzes code and reports potential errors and bad practices
    @typescript-eslint/parser -> Allows ESLint to understand and parse TypeScript syntax
    @typescript-eslint/eslint-plugin -> Adds TypeScript-specific linting rules to ESLint
    commander -> Creates and manages command-line interface (CLI) commands and options
    dotenv -> Loads environment variables from a .env file into process.env
    winston -> Provides structured logging with support for files, consoles, and log levels
    chalk -> Adds colors and styles to terminal output
    ora -> Displays loading spinners and progress indicators in the terminal
    @prisma/client -> Runtime database client generated by Prisma for database operations
    prisma -> Prisma CLI used for schema management, migrations, and client generation
